# TRANSLATOR
## DESCRIPTION
This project is a simple yet functional Language Translator GUI application built using Python, Tkinter, and Googletrans API. The app allows users to enter text in one language and translate it into another using an intuitive graphical interface.
## FEATURES
✔️ User-friendly GUI created with Tkinter

✔️ Supports translation between multiple languages (powered by googletrans)

✔️ Dropdown menus to select source and destination languages

✔️ Multi-line text boxes for easy input and output

✔️ “Translate” button to perform instant language translation

✔️ Smooth error handling for invalid inputs or API issues
## HOW TO RUN THE PROGRAM
The user types or pastes text into the Source Text box.

They choose the source language and destination language from the dropdown menus.

When the Translate button is clicked, the app uses the googletrans library to translate the input text.

The translated text appears in the Destination Text box.
