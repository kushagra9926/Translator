## Project Title

**Translator**

## Overview of the Project

This project is a simple Python-based language translation application built using Tkinter for the graphical user interface and Googletrans for translation functionality. The program allows users to input text, select the source and destination languages, and instantly receive the translated output. It demonstrates GUI development, third-party API integration, and basic error handling.

## Features

* Clean and user-friendly Tkinter GUI
* Supports translation between multiple languages
* Dropdown menus for language selection
* Multi-line text input and output fields
* Real-time translation using Googletrans
* Error handling for invalid inputs and translation failures

## Technologies Used

* **Python 3.x**
* **Tkinter** – for the graphical user interface
* **Googletrans** – for translation services
* **ttk Widgets** – for themed interface elements

## Steps to Install and Run the Project

### 1. Clone the Repository

```bash
[git clone https://github.com/your-username/python-language-translator-gui.git
cd python-language-translator-gui](https://github.com/kushagra9926/Translator.git)
```

### 2. Install Dependencies

```bash
pip install googletrans==4.0.0-rc1
```

(Tkinter comes pre-installed with most Python distributions.)

### 3. Run the Application

```bash
python translator.py
```

## Instructions for Testing

* **Functional Testing:**

  * Enter text in the input box and verify translation.
  * Test with multiple language combinations.
  * Check behavior when the input box is empty.

* **Error Handling:**

  * Disconnect Internet and verify that the program shows an appropriate error message.
  * Enter unsupported or special characters to see how the program behaves.

* **Performance Testing:**

  * Try translating long paragraphs.
  * Click the Translate button repeatedly to ensure stability.

* **User Interface Testing:**

  * Ensure dropdown menus work correctly.
  * Verify that text boxes accept and display text properly.
